# Privacy Policy

DreamCatcher is a personal desktop application for reviewing Oura sleep data.

## Data Use

DreamCatcher uses Oura account data only to display sleep information for the authenticated user inside the app.

## Storage

OAuth tokens are stored locally on the user's device through the operating system credential store. Cached sleep data is stored locally on the user's device.

## Sharing

DreamCatcher does not sell, share, or transmit user sleep data to third parties beyond the Oura API requests required to fetch the user's own data.

## Contact

This project is provided as a personal-use software project through its GitHub repository.
